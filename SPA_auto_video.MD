# SPA 자동 영상 생성 도구

## 목적

SPA(Single Page Application) 웹페이지를 자동으로 녹화하고,
편집 없이 바로 사용할 수 있는 영상 파일(mp4 등)로 변환하는 도구를 만든다.

## 주요 기능 (예정)

- [ ] 지정한 URL의 SPA 페이지 자동 녹화
- [ ] 페이지 내 특정 인터랙션(클릭, 스크롤 등) 시나리오 재생 후 녹화
- [ ] 녹화본을 mp4/gif 등으로 변환
- [ ] 해상도/프레임레이트 옵션 설정

## 기술 스택 (검토 중)

- 브라우저 자동화: Playwright 또는 Puppeteer
- 영상 인코딩: ffmpeg
