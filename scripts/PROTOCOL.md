# 주간 랜덤 대본 생성 프로토콜

Day1~Day5로 기초 난이도 램프(SPA 33 → 75)를 완료한 이후, 매주 새로운 주제로
대본을 하나씩 무작위 생성하기 위한 규칙입니다. "이번 주 대본 만들어줘"라고
요청하면 아래 절차를 그대로 따릅니다.

## 1. 기본 구조 (고정)

- 총 러닝타임: 20분
- Part 1~5, 각 4분씩 균등 배분
- 순서 고정: Personal → Opinion → Picture → Graph → Article
- Part1, 2는 질문 3개 / Part3, 4, 5는 질문(또는 지문) 1개 + 모범답안

## 2. 난이도 기준 (Day1/3/5를 앵커로 사용)

| 난이도 | 기준 대본 | 특징 |
| --- | --- | --- |
| Easy | [day01.md](day01.md) | 짧고 단순한 문장, 기초 어휘 |
| Intermediate | [day03.md](day03.md) | 접속어·근거·반례 포함, 중간 길이 문장 |
| Advanced | [day05.md](day05.md) | 양보/반박 구문, 관용 표현, 원어민식 자연스러운 어투 |

기본값은 가장 최근에 도달한 난이도를 유지합니다. 목표 점수가 오르면 다음 단계로 상향합니다.

## 3. 주제 무작위 선정 규칙

각 Part마다 아래 주제 풀에서 무작위로 하나씩 고릅니다. 단, [used-topics.md](used-topics.md)에
기록된 **최근 4주 이내 사용 주제는 제외**합니다.

### Part 1. Personal Questions
업무 경험/성취/실수 · 워라밸/스트레스 관리 · 동료 관계/팀워크 · 자기계발/목표 · 일상 습관/취미

### Part 2. Opinion Questions
기술·AI와 일자리 · 리더십 스타일 · 재택근무 vs 사무실 근무 · 개인 성과 vs 팀워크 평가 · 사회·경제 이슈(세계화, 환경 등)

### Part 3. Describe a Picture
사무실 회의 · 공장/생산 현장 · 컨퍼런스/발표 · 스타트업 사무실 · 고객 응대/서비스 현장

### Part 4. Analyze a Graph
매출/판매량 추이 · 시장 점유율 비교 · 산업별 트렌드 변화 · 지역별 비교 · 연도별 성장/감소 추세

### Part 5. Summarize an Article
산업 트렌드(전기차, AI, 재생에너지 등) · 조직문화/원격근무 · 경제/정책 이슈 · 리더십/경영 전략 · 사회적 이슈(글로벌화, 웰빙 지표 등)

## 4. 생성 절차

1. `used-topics.md`에서 최근 4주 사용 주제를 확인하고 제외한다.
2. 5개 Part 주제 풀에서 각각 무작위로 1개씩 고른다.
3. 지정된 난이도 기준에 맞춰 질문/지문 + 모범답안을 작성한다.
4. 파일명은 `scripts/weekNN.md` (NN은 2자리 순번, 예: `week01.md`)로 저장한다.
5. 파일 상단에 아래 메타 정보를 기록한다.

   ```
   # Week NN - [난이도] (생성 날짜)

   - 난이도: Easy / Intermediate / Advanced
   - 사용 주제: Part1(...), Part2(...), Part3(...), Part4(...), Part5(...)
   ```

6. `used-topics.md`에 이번 주 사용한 주제를 한 줄 추가한다.
7. `../SPA_auto_video.MD`의 진행 계획 표에 새 줄을 추가한다.

## 5. 호출 방법

"이번 주 대본 만들어줘" / "Week N 대본 만들어줘"라고 요청하면 위 절차대로
새 `scripts/weekNN.md` 파일을 생성합니다. 특정 난이도를 원하면 "Advanced로
만들어줘"처럼 지정할 수 있습니다.
